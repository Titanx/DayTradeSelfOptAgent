# Agents 模块

多智能体核心，实现分析师→研究员→交易员→风控的完整投研决策链。

## 子模块

| 模块 | 说明 |
|------|------|
| `analysts/` | 四位分析师，分别从基本面、技术面、情绪面、政策面收集信息 |
| `researchers/` | 多空研究员，基于分析师报告进行多方/空方辩论 |
| `utils/` | 共享工具，包括数据读取工具和交易记忆系统 |
| `schemas.py` | Pydantic 数据模型，定义各环节的结构化输出格式 |
| `trader.py` | 交易员 Agent，基于研究员结论生成符合 A 股 T+1 规则的交易提议 |
| `risk_mgmt.py` | 风控模块，三方辩论（激进/保守/中性）+ 组合经理最终评级 |

## 数据流

```
分析师报告 → 研究员辩论 → 交易员提议 → 风控辩论 → 最终评级
```
